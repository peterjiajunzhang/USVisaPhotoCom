campaign.html 按导出尺寸固定为 1284×2778，不能直接塞进窄 iframe。这里用「固定尺寸 + transform: scale + 裁切框」缩成五列同一行，可看到完整 slide。屏宽不够时左右滑动这一行即可。改完 campaign 后整页刷新本预览；加载时会为 iframe 追加时间参数，避免沿用缓存里的旧 HTML。单独打开 campaign.html 时可用强制刷新（macOS：⌘⇧R）。定稿后在目录运行 npm run export 出 PNG。
